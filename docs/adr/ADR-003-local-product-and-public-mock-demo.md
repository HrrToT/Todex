# ADR-003：本地真实产品与线上 Mock 演示站分离

状态：approved
日期：2026-07-13

## 决定

Electron 桌面端是唯一可以访问用户本地工作区和真实 LLM 凭据的产品宿主。公网 WebUI 仅运行内置 Node.js/Python 示例仓库和 Mock LLM，不接收真实 API Key，也不提供自由 shell。

## 理由

公网服务无法安全、自然地获得访问者本地文件系统权限。分离后既满足课程要求的可访问 WebUI URL，也不会为了云端访问本地仓库而引入代理、账户、同步或高风险凭据存储。

## 后果

线上站的功能范围受限，但它可以稳定、无网络依赖地展示危险动作拦截、反馈回灌和自修复的确定性机制。
